17) Padding: Padding is the space between the content of an element and its border.
Padding can be set using properties like padding-top, padding-right, padding-bottom, and padding-left, or shorthand with padding.

Margin: Margin is the space outside of an element, between the element's border and adjacent elements.
Margin can be set using properties like margin-top, margin-right, margin-bottom, and margin-left, or shorthand with margin.

19)

em: The em unit is relative to the font size of its parent element.
ex- if the font size of a parent element is 16px and font size of a child element is 1.5em, it will be 24px

rem: The rem unit is relative to the font size of the root (html) element.
ex- if the font size of the root element is 16px the font size of an element 1.5rem, it will be 24px
